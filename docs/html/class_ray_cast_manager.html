Realiza un raycast y devuelve el primer Collider impactado con etiquetas especificadas.

Parámetros
rayEl rayo a lanzar.
entityManagerReferencia al EntityManager para acceder a los Colliders.
maxDistanceDistancia máxima que el rayo debe recorrer (predeterminado: infinito).
tagsVector de etiquetas para filtrar impactos (predeterminado: vacío, impacta todo).
Devuelve
Un std::optional<RaycastHit> si se detecta una colisión con una etiqueta coincidente, std::nullopt en caso contrario.
