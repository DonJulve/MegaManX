\doxysubsection{Descripción detallada}
Gestor de audio para manejo de efectos de sonido y música. 

\doxylink{class_audio_manager}{Audio\+Manager} implementa el patrón singleton para proporcionar acceso centralizado a las funcionalidades de carga y reproducción de audio utilizando SDL\+\_\+mixer.
